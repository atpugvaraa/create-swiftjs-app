cli tool written used by swift.js to create a react/nextjs based web-app

Dependencies:
- [golang](https://go.dev/)
- [bun](https://bun.sh/)

Build and Run steps:
1. Clone repo
2. run in terminal
```sh
go mod tidy
```
```sh
go build -o create-swiftjs-app main.go
```
6. Usage:
```sh
./create-swiftjs-app new
```

You have a working create-swiftjs-app!
